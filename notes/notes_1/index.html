Tutorial of Doc-site Deployment with Hugo and Github Pages

Description: Test of Stone

Contents

  • Create a Documentation Site in Hugo Theme Khata

Install Hugo on Windows 10

  1. Download the according Hugo zip file for Windows 10 from Hugo Releases.
  2. Extract the zip file and you will see one executable and a couple markdown files.
  3. Copy hugo.exe to root folder, i.e. C:\Hugo\bin.
  4. Open Control Panel, find All Control Panel Items and then System; add your Hugo path (i.e.C:\Hugo\bin) to your Environment Variables as both in User Variables and System Variables; click OK to all buttons.
  5. Test your Hugo from cmd C\users\username\>Hugo Version
    If succeed, recive codes:
    v0.81.0-59D15C97 windows/amd64 BuildDate=2021-02-19T17:07:12Z VendorInfo=gohugoio

Create a Documentation Site in Hugo Theme Khata

  1. Create a new website in Microsoft Visual Studio(MVS)’s terminal: hugo new site stone_doc
  2. Download Khata theme in MVS
cd C:\Users\stone\stone_doc\themes
$ git clone https://github.com/darshanbaral/khata.git
  1. Initiallise your page on local server:
cd PS C:\Users\stone\stone_doc> hugo server
Start building sites
... ...
Web Server is available at //localhost:1313/ (bind address 127.0.0.1)
Press Ctrl+C to stop

Organise Page Content with Microsoft Visual Studio Markdown Editor

  1. Dwoanload Microsoft Visual Studio (MVS) and install Markdown Editor
  2. Edit file notes_1.md with this dictorial content
  3. Edit Homepage content in Config.toml, Change baseUrl to https://github.com/stone5508/stone5508.github.io.git

Hosting onto Github Pages

  1. Create a stone5508.github.io repository on GitHub.
  2. Below the root directory of stone_doc enter
hugo --theme=khata --baseUrl="https://github.com/stone5508/stone5508.github.io.git"
cd public
git init
git add .
git commit -m "my first commit"
git remote add origin https://github.com/stone5508/stone5508.github.io.git
git push -u origin master
  1. Open https://github.com/stone5508/stone5508.github.io, click Setings and enjoy your hosted site of Github Pages
